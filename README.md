# #2 Problems solved by GraphQL
- GraphQl로 해결할 수 있는 두 가지 문제
    - Over-fetching : 요청한 영역의 정보보다 많은 정보를 서버로부터 받는 것
    - Under-fetching : 하나의 작업을 위해 많은 소스를 여러 번 요청하는 것
    ```
    하나의 Query로 필요한 정보만 가져올 수 있다.
    ```
